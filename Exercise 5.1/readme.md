## Average Marks Calculation

# Instruction

Write a program to calculate average marks available in the list.

e.g. student_marks = [78, 85, 65, 92, 88]

# Average Formula

![Screenshot](agv.png)

  408 / 5 = 81.6

</pre>

# Example Output
