## BMI Calculator

# Instruction

Write a program to get input from user his height in meters and his weight in Kg. to calculate and display his BMI(Body Mass Index)

![Screenshot](bmi.png)

**Be Carefull:-** You have to round the result to nearest whole number.

# Example Input

weight = 70

height = 1.68

# Example Output

70 / 1.68 \* 1.68 = 24.801587301

24
