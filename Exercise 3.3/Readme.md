# Instruction

A leap year is a year that has an extra day, February 29, making it 366 days long instead of the usual 365. Leap years are necessary to keep our calendar in alignment with the Earth's revolutions around the Sun, which takes approximately 365.2422 days.

To determine if a year is a leap year in the Gregorian calendar (the most widely used calendar system), you can follow these rules:

- If the year is evenly divisible by 4, it's a leap year.
- However, if the year is evenly divisible by 100, it's not a leap year, unless:
- If the year is also evenly divisible by 400, then it is a leap year.

# Example

e.g. Year 2004

- 2004 / 4 = 501(Leap)
- 2004 / 100 = 20.04(Leap)
- 2004 / 400 = 5.01(Not Leap)

So the Year 2004 is Leap Year
