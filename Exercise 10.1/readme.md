### Days in Month

# Instruction

In the main.py you will find the solution for leap year challange exercise.

First convert this to function **_`is_leap()`_** so as to printing "Leap Year" or "Not a Leap Year" it should return **_`True`_** or **_` False`_**.

Then create a function called **_`days_in_month()`_** which will take input a year and month as input.

e.g.

**_`days_in_month(year=2024, month=2)`_**

And it will use this information to work out the number of days in the month, then return that as output

e.g.

**_`28`_**

The List month_days contain the number of days in a month from Jan to Dec. for non leap year. A leap year has 29 days in Feb.
